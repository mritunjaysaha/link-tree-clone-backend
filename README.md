# link-tree-clone-backend

### Install

1. Install dependencies

```bash
npm install
```

2. Install `nodemon`

```bash
npm i nodemon
```

### Set environment variables

```env
MONGO_URI=
SECRET=
```

### Run

```bash
npm start
```

or

```bash
npm run dev
```

The server will be running of `http://localhost:9000`
